# Wesley College - LMS
Wesley college - lms app create by local data.
---

### To Run Locally

```bash
git clone https://github.com/kacMadushan/wesley-college-lms.git
cd wesley-college-lms
yarn install
yarn build:dev
yarn start
Open http://localhost:8080 to view it in the browser.
```
